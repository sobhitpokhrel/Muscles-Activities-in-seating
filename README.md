# Muscles-Activities-in-seating
Muscles load under vibration : Suspension vs Non-Suspension Seats

# Design of Experiment
Suspension vs Non-Suspension Seats
One participant, two seats, 15 minutes of dynamic vibration (One direction only)\


# Electromyography (EMG) sensors placement
SCM muscles – Left Right
Trapezius Muscles – Left Right
Trapezius muscles – Left Thoracic Area
Lumbar Erector Spinae – Left Right
Latissimus dorsi – Left Right

Data Processing:
Raw Data > RMS value > Comparisions using Matlab
